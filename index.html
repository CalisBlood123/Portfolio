Sobre mí

Soy un desarrollador web en formación apasionado por el diseño y la creación de páginas web modernas, responsivas y funcionales. ¿Que puedo hacer?

  • Utilizar Bootstrap para agilizar el desarrollo.
  • Aplicar buenas prácticas de HTML y CSS.
